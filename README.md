# Artist Browser

Discover 5,819 artists across 34 genres. Click any artist to find playlists on Spotify.

## Features

- **34 Genres** - From Electronic to Hip Hop, Jazz to Metal
- **5,819 Artists** - Curated selection across all genres
- **Instant Search** - Find any artist as you type
- **One-Click Spotify** - Opens playlists directly in Spotify app
- **No Login Required** - Works immediately
- **Mobile Friendly** - Responsive design
